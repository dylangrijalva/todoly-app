## Tech Stack

* Backend: Node.js, Express.js, MySQL, and Docker.

* Basic CRUD operations: `create`, `read`, `update`, and `delete` tasks.

## Run locally

Clone the project

```bash
    git clone https://github.com/dylangrijalva/todoly-app
```

Backend app:

```bash
    cd backend/ && bash run.sh
```
